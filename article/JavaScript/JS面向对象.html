# 面向对像

通过类可以创建任意多个具有相同属性和方法的对象。
由于js中没有类的概念，js的对象与基于类的语言中的对象有所不同

特点：

  1. 把功能写成一个函数
  2. 函数必须在对象身上，调用 对象.方法名()
  3. 对象身上的功能可以继承

# 创建对象的方式

# 字面量方法

let person = {
    name:"xu",
    age:22,
    job: "web"
}

# Object构造函数模式

let obj = new Object()

# 工厂模式

function createPerson(a,b,c){
    let o = new Object;
    o.name = a;
    o.age = b;
    o.job = c;
    o.sayName = function(){
        alert(this.name)
    }
}
let person = createPerson("xu",22,"web")

# 构造函数模式

function Person(a,b,c) {
    this.name = a;
    this.age = b;
    this.job = c;
    this.sayName = fucntion(){
        alert(this.name)
    }
}
let person = new Person("xu",22,"web");

# new操作符原理

  1. 创建一个空对象
  2. 将这个空对象的prototype指向构造函数的原型
  3. 将这个构造函数的作用域赋给新对象(即this指向这个新对象)
  4. 执行构造函数内部的代码
  5. 返回这个新对象
# 构造函数的问题

每次new一个新对象，内部的函数也会新建一个，无法共享，造成浪费

# 原型模式

function Person() {

    Person.prototype.age = 22,
    Person.prototype.job = "web",
    Person.prototype.sayName = function(){

    }
}
let person = new Person();

# 原型对象

无论何时，只要创建一个新函数，js就会为该函数创建一个prototype属性，该属性指向该函数的原型对象。而原对象会自动获得一个constructor属性(构造函数属性)指向该函数。
如下图

原型对象默认只会取得constructor属性，至于其他方法则继承自Object
当调用构造函数创造一个新实例后，该实例内部将包含一个指针(内部属性)[[prototype]。按理来说[[prototype]]是访问不到的，但是浏览器中提供了一个访问的方法_proto_ 。

每个实例和构造函数没有直接的关系，连接存在于实例和构造函数的原型对象之间

# 原型对象API

  • idPrototypeOf()
  • Object.getPrototypeOf()
  • hasOwnProperty()
  • in 操作符
  • Object.key()
# isProtypeOf()

来确定对象是否指向同一个原型

Person.prototype.isPrototypeOf(person1)    // true
Person.prototype.isPrototypeOf(person2)    // true
# Object.getPrototypeOf()

这个方法返回[[prototype]]的值
近似于_proto_.

Object.getPrototypeOf(person1) = Person.prototype
Object.getPrototypeOf(person1).name = "xu"
# hasOwnProperty()

继承自Object
可以检测一个属性是否存在于实例中，还是存在于原型中
实例属性还是原型属性

function Person(){
    Person.prototype.name = "xu"
}
let person1 = new Person()
let person2 =new Person()

person1.name // "xu"
person1.hasOwnProperty("name")   // false
person1.name = "xu"
person1.hasOwnProperty("name")  // true
# in操作符

只要对象能访问到的属性就返回true，无论是实例属性还是原型属性

function Person(){
    Person.prototype.name = "xu"
}
let person1 = new Person()

console.log("name" in person1)  // true
person.name = "yu"
console.log("name" in person1)  // true

# Object.keys()

取得对象上所有可枚举的实例属性
接受一个对象参数，返回一个包含所有可枚举属性的字符串数组

function Person(){
    Person.prototype.name = "xu",
    Person.prototype.age = 22
}

console.log(Object.keys(Person.prototype))    // "xu,22"

let person1 = new Person()
person1.name = "yu"
person1.age = 23
console.log(Object.keys(person1))   // "yu,23"
